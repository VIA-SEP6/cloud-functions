# Cloud Functions Notifications

***

### `POST` Mark Notification As Read

Mark notification as read<br>

`notifications-markAsRead`

```js
const requestBody = {
    data: {
        notificationId: "46345",
    },
};
```

***

`notifications-markAllAsRead`
